# Bakery-Order-and-Custom-Cake-Booking-Platform
# Online Bakery Store

A Spring Boot based online bakery ordering platform with customer login, custom cake booking, product management, order management, and admin analytics.

## Features

- Customer registration and login
- User profile management
- Product browsing and custom cake ordering
- Order placement with delivery details
- Admin dashboard for managing:
  - users
  - products
  - orders
  - analytics
- Inventory and revenue overview
- Responsive front-end pages
- File-based storage for project data
- Ready for deployment with Docker

## Tech Stack

- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS
- JavaScript
- Maven

## Project Structure

- `src/main/java` - backend controllers, models, and services
- `src/main/resources/templates` - Thymeleaf pages
- `src/main/resources/static` - CSS, images, and static assets
- `src/main/resources` - application configuration and data files

## Requirements

- Java 17+
- Maven 3.9+
- Browser
- Git
